Creativity

Creativity can be defined as the ability to generate novel, and valuable, ideas (Boden, 2009).

Novel

  • P-creativity: new to the person who generated it
  • H-creativity: new to the world!

Valuable

  • Spectrum from casual creativity to power users — from doodles to masterpieces — requiring different trade-offs in tools

Computational creativity

Algorithmic or generative systems which can automate part, or all, of the creative act by formal, computational means.

Machine learning can be used if it is desirable for the system to 'learn' by using data to improve performance

Scenario

Create a device which mimics a vinyl 'scratch' sound, for performance with a mouse, mousepad, or tablet.
